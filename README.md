# Behind the Interface

A critical-thinking video game prototype — a 2D top-down investigation set on a university campus.

> Day 1 vertical slice. Build target: a 30-minute, five-day mystery that uses gameplay (not quizzes) to teach the five critical-thinking tools.

**Play it:** [https://&lt;your-username&gt;.github.io/&lt;repo&gt;/](#)

---

## What it is

You're an undercover detective posing as a student. A peer has been found dead in the auditorium. The "obvious" answer is revenge; the truth is something else entirely. Reaching the correct conclusion takes the five critical-thinking tools — asking meaningful questions, analysing logic, spotting bias, considering multiple viewpoints, and evaluating evidence.

This prototype covers **Day 1**: arrival, the Forum, the crime scene, and the captain's briefing. About 5 minutes of play.

## How to play

| Key | Action |
|---|---|
| `WASD` / arrow keys | Walk |
| `E` | Interact with people, doors, and items |
| `Space` / `Enter` / `E` | Advance dialogue |
| `1` – `9` | Pick a dialogue option |

## Status

Vertical slice. One day, three rooms (Outside, Forum, Auditorium), six NPCs, one ending. Built in vanilla HTML / CSS / JavaScript — no build step.

The full vision is five days, seven rooms, three endings, and a structured post-game debrief that is the real pedagogical payload.

## Project

Final-year co-creators project at the **University of Westminster** — a critical-thinking module taught through a video game instead of a textbook. The single playable level here is the synthesis (boss) level of an eventual six-level curriculum.
